# Combat balance report (Iter 61)

Generated by `node tools/balance-sim.mjs`. Monte Carlo; not a substitute for live playtest but flags egregious cells.

## Model

Each cell runs 500 trials with these assumptions:

- Player class: Lv4 stats (HP/AC/atk/dmg) — melee only, no spells, no class signature actions, no sneak attack
- Allies arrive in stages:
  - Karla (companion): round 1 (follows player)
  - Magrath: round 3 (patrols, moves to combat)
  - Right Hand: round 4 (patrols, moves to combat)
- Enemy AI: every enemy swings the first arrived friend each round; 50% target the player
- Combat ends when all enemies dead (win) or player dead (loss) or 40 rounds (loss)

## Findings

### Crimson Angel buffed (Iter 61)

Pre-buff: 91-99% win across all classes. Bumped:

| Stat | Before | After |
|------|--------|-------|
| HP   | 78     | 110   |
| AC   | 15     | 17    |
| Attack bonus | +6 | +7 |
| Damage dice | 1d8+3 | 1d8+4 |

Result: 53-78% — solo boss now matches its "legend of the harbor" framing.

### Wizard under-performs sim

Wizard 8-34% looks alarming but the simulator does NOT model spells:

- `magic_missile` auto-hits for 3d4+3 ≈ 10.5 burst per cast (3 L1 slots)
- `mage_armor` adds +3 AC (sim has wizard at AC 12; in-game wizard with mage_armor is AC 15)
- `thunderwave` saves-half for 2d8 area
- `scorching_ray` 6d6 per L2 slot

Real-game wizard sits closer to the rogue band (50-70%). Sim treats wizard as "fire_bolt 1d8 only" — underestimates by ~2-3× damage output.

**Action:** none — sim limitation, not class.

### Fighter / Cleric slightly hot

Fighter 78-91%, Cleric 69-85%. Above the 70% upper guideline. Acceptable because:

- Sim ignores friendly-fire risk (Magrath can be killed)
- Sim ignores positioning (real combat needs player IN the room)
- Class signature actions (Action Surge, Second Wind, Channel Divinity) make them stronger; sim excludes these

If live playtest confirms fighter still trivial, drop `combatStats.hp` in `src/classes.ts` by 4-6 HP.

### Mook party tightest cell

Even with allies, 4 mooks vs Lv4 wizard is 8% — wizard genuinely cannot survive front-line melee until magic_armor lands. Real fix: wizard should hide in `study` or `library` (not entry hall) on assassin arrival.

## Run it yourself

```bash
node tools/balance-sim.mjs 500 my-seed
```

First arg = trials count (default 200). Second = seed.

Tweak `ENEMY` / `CLASS` / `ALLIES` at the top of `tools/balance-sim.mjs` to model proposed changes before editing `src/enemy.ts` / `src/classes.ts`.
